Input commands are to be written as new lines into instructions_input.txt

If commands include additional arguments, they should be on the same line 
following the command and separated by commas. 
For example, the below command has three arguments:

SomeCommand,SomeArgument,Another Argument,AThirdArgument

--sent after acquisition of single image stack complete--
command: AcquisitionDone
arguments: None
example: AcquisitionDone

--sent after movement to new coordinates is complete--
command = StageMoveDone
arguments: x, y, z
example: StageMoveDone,1,34.2,-2

--sent after request for current position--
command = CurrentPosition
arguments: x, y, z
example: CurrentPosition,2,0,-3.2

--sent after uncaging is finished--
command = UncagingDone
arguments: None
example: UncagingDone

--sent to establish Field of View size in Âµm (when completely zoomed out)--
command = FovXYum
arguments = X,Y
example: FovXYum,250,250

--sent when new zoom is set--
command = Zoom
arguments = int (zoom value)
example: Zoom,25

--Sent when new galvo scanning voltage is set
command = ScanVoltageXY
arguments = x,y
example: ScanVoltageXY,0.02,1.2

--sent when scan voltage multiplier is requested
command = ScanVoltageMultiplier
arguments = X, Y
example: ScanVoltageMultiplier,5,15

--sent when scan voltage range reference is requested
command = ScanVoltageRangeReference
arguments = X, Y
example: ScanVoltageRangeReference,5,15

--sent when number of Z slices is requested to change
command = ZSliceNum
arguments = number of slices
example: ZSliceNum,10

--Sent when intensity file path is requested
command = IntensityFilePath
arguments = filepath
example: IntensityFilePath,c:/imaging/image1.tiff

--Sent when imaging resolution is requested
command = ResolutionXY
arguments = x,y
example: ResolutionXY,512,512

--Sent when intensity saving is turned on/off
command = IntensitySaving
arguments = 1/0 (on/off)
example: IntensitySaving,1



Output commands are written line by line to instructions_output.txt

Commands may include additional arguments, just like input commands

--Move to new stage position--
command = SetMotorPosition
arguments: x y z
example: SetMotorPosition,12,89.2,0
expected return command: StageMoveDone

--Grab a stack once--
command = StartGrab
arguments: None
example: StartGrab
expected return command: AcquisitionDone

-- Turn intensity file saving on/off--
command = SetIntensitySaving
arguments: 1 or 0 (on or off)
example: SetIntensitySaving,1
return: IntensitySaving

-- Get Path of Intensity File--
command = GetIntensityFilePath
arguments: None
example: GetIntensityFilePath
return: IntensityFilePath

--Set Zoom to specific value
command = SetZoom
arguments: zoom
example: SetZoom,20
expected return command: Zoom

--Do uncaging--
command = StartUncaging
arguments: ROI_x_pixels, ROI_y_pixels
example: StartUncaging,37,42
expected return command: UncagingDone

--Request for current position to be sent--
command = GetCurrentPosition
arguments: None
example: GetCurrentPosition
expected return command: CurrentPosition

--Request for FOV size--
command = GetFOVXY
arguments: None
example: GetFOVXY
expected return command: FovXYum

--Request Galvo Scanning Position--
command = GetScanVoltageXY
arguments: None
example: GetScanVoltageXY
expected return command: ScanVoltageXY

--Set Galvo Scanning Position
command = SetScanVoltageXY
arguments: x,y
example: setScanVoltageXY,0.2,-4
expected return command: ScanVoltageXY

--Request Galvo Scan Voltage Multiplier
command = GetScanVoltageMultiplier
arguments: None
expected return command: ScanVoltageMultiplier

--Request Galvo Scan Voltage Range Reference
command = GetScanVoltageRangeReference
arguments: None
expected return command: ScanVoltageRangeReference

--Set Number of Z Slices
command = SetZSliceNum
arguments: zSliceNum
example: SetZSliceNum,10
expected return command: ZSliceNum

--Set X,Y Imaging Resolution
command = SetResolutionXY
arguments: x,y
example: SetResolutionXY,128,128
expected return command: ResolutionXY

--Get X,Y Imaging Resolution
command = GetResolutionXY
arguments: None
example: GetResolutionXY
expected return command: ResolutionXY
